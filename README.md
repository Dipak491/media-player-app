# Media Player App

A modern, feature-rich media player application built with a focus on usability, performance, and extensibility. This project provides a clean interface to play audio and video files, manage playlists, and customize playback settings.

## Features

- **Audio & Video Playback**: Supports major media formats.
- **Playlist Management**: Create, edit, and save playlists.
- **User-Friendly Interface**: Intuitive UI for seamless navigation.
- **Playback Controls**: Play, pause, seek, volume, repeat, shuffle, etc.
- **Media Metadata**: Display song/video information like title, artist, album, etc.
- **Custom Themes**: Switch between light and dark modes.
- **Keyboard Shortcuts**: Quick control over playback.
- **Extensible**: Easily add plugins or new features.

## Getting Started

### Prerequisites

- Node.js (v14 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/Dipak491/media-player-app.git
    cd media-player-app
    ```

2. Install dependencies:
    ```bash
    npm install
    # or
    yarn install
    ```

3. Start the development server:
    ```bash
    npm start
    # or
    yarn start
    ```

4. Open the app in your browser or run the executable if building for desktop.

### Building for Production

To build the application for production:
```bash
npm run build
# or
yarn build
```

## Usage

- Launch the app.
- Use the file picker or drag-and-drop to add media files.
- Manage your playlists and adjust settings as needed.
